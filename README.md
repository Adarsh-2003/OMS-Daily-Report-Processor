# OMS Daily Report Processor

A simple web-based tool to process and transform Duke Energy data for Cognizant reporting.

## What it does

- Extracts job names from short descriptions
- Processes priority levels (converts "Moderate" to "Medium")
- Filters and reorganizes columns
- Maintains tabular format for easy copy-paste

## How to use

1. Open `index.html` in your web browser
2. Enter your name in the "Filled By" field
3. Paste your tabular data from Excel or the source sheet
4. Click "Process Data"
5. Copy the processed output to your destination sheet

## Features

- Dark-themed, user-friendly interface
- No installation required - works in any browser
- Preserves tabular format for easy copying
- Responsive design for different screen sizes

## Technical details

Built with vanilla HTML, CSS, and JavaScript. No dependencies or external libraries required.
